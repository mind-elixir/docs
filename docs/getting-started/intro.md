# 简介

Mind elixir is a free open source mind map core.

- Lightweight
- High performance
- Framework agnostic
- Pluginable
- Export as SVG or PNG
- Build-in drag and drop / node edit plugin
- Summarize nodes
- Bulk operations supported
- Undo / Redo
- Efficient shortcuts
- Styling your node with CSS
