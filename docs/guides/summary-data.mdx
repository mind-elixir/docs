## 摘要数据

```typescript
export interface Summary {
    id: string;
    text: string; 
    parent: string; 
    start: number; 
    end: number;
}
```

摘要数据包含以下字段：

- `id`: 唯一 id
- `text`: 摘要文本
- `parent`: 摘要所属节点 id
- `start`: 摘要开始的子节点索引
- `end`: 摘要结束的子节点索引

:::warning

摘要的父节点不能是根节点。

:::
