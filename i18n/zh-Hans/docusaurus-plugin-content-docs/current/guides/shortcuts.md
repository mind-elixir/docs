# 快捷键

| 快捷键             | 功能                   |
| ------------------ | ---------------------- |
| Enter              | 插入兄弟节点           |
| Shift + Enter      | 向前插入兄弟节点       |
| Tab                | 插入子节点             |
| Ctrl + Enter       | 插入父节点             |
| F1                 | 居中思维导图           |
| F2                 | 编辑当前节点           |
| ↑                  | 选择上一个节点         |
| ↓                  | 选择下一个节点         |
| ← / →              | 选择左右侧的节点       |
| PageUp / Alt + ↑   | 上移                   |
| PageDown / Alt + ↓ | 下移                   |
| Ctrl + ↑           | 使用两侧分布的布局方式 |
| Ctrl + ←           | 使用左侧分布的布局方式 |
| Ctrl + →           | 使用右侧分布的布局方式 |
| Ctrl + C           | 复制                   |
| Ctrl + V           | 粘贴                   |
| Ctrl + "+"         | 放大思维导图           |
| Ctrl + "-"         | 缩小思维导图           |
| Ctrl + 0           | 重置大小               |
