# 快捷键

使用右键菜单操作节点带来诸多不便，大大降低了工作学习效率，Mind Elixir 支持更高效的快捷键，让你轻松流畅地创建图表并浏览思维导图！这些快捷键使浏览和探索复杂的思维导图变得更加便捷和直观。

| 快捷键             | 功能                   |
| ------------------ | ---------------------- |
| Enter              | 插入兄弟节点           |
| Shift + Enter      | 向前插入兄弟节点       |
| Tab                | 插入子节点             |
| Ctrl + Enter       | 插入父节点             |
| F1                 | 居中思维导图           |
| F2                 | 编辑当前节点           |
| ↑                  | 选择上一个节点         |
| ↓                  | 选择下一个节点         |
| ← / →              | 选择左右侧的节点       |
| PageUp / Alt + ↑   | 上移                   |
| PageDown / Alt + ↓ | 下移                   |
| Ctrl + ↑           | 使用两侧分布的布局方式 |
| Ctrl + ←           | 使用左侧分布的布局方式 |
| Ctrl + →           | 使用右侧分布的布局方式 |
| Ctrl + C           | 复制                   |
| Ctrl + X           | 剪切                   |
| Ctrl + V           | 粘贴                   |
| Delete / Backspace | 删除节点/箭头/摘要     |
| Ctrl + "+"         | 放大思维导图           |
| Ctrl + "-"         | 缩小思维导图           |
| Ctrl + 0           | 重置缩放               |
| Ctrl + K, Ctrl + 0 | 折叠所有节点           |
| Ctrl + K, Ctrl + = | 展开所有节点           |
| Ctrl + K, Ctrl + 1-9 | 展开到第N级          |
