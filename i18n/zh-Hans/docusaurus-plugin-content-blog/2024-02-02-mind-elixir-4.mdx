## 新特性

- 更丰富的色彩，带来更强的印象感
- 比 3.x 更完整的多节点操作体验
- 图片输出功能更新
- 快捷键更新
- 支持插入自定义节点
- 更完善的文档[已上线](https://docs.mind-elixir.com/)

## 从 3.x 升级到 4.x

3.x 到 4.x 中，linkData的数据格式作出了修改，请注意数据迁移。另外有少量 API 调整，详情见[此链接](https://github.com/SSShooter/mind-elixir-core/wiki/Breaking-Change#400)。

## 其他特性

- 0 依赖，高速安装（33kb gzip 后）
- 高性能节点操作
- 不与前端框架绑定
- 可使用插件
- 多节点总结
- 节点关联 
- 撤销与重做
- 可以自定义主题，让你的节点更多彩
- 支持一定的 CSS 样式修改

## 思维导图的特性

- 提倡精简的节点主题，归纳出关键词
- 简单的关键词更容易触发联想
- 色彩加强印象，更容易记忆
- 自由收缩展开，信息更清晰
- 树形结构节点关系更清晰
- 总结、关联等非线性表达
