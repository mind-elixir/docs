# サマリー

Mind Elixirはマインドマップにノードのサマリーを追加することができます。以下の例をご覧ください：

<MindElixirShowcase
    height='380px'
    data={{
        nodeData: 
            {
                id: 'root',
                topic: 'Root',
                children: [
                    {
                        topic: 'Child 0',
                        id: 'child0',
                        children:[
                    {
                        topic: 'Child 1',
                        id: 'child1',
                    },
                    {
                        topic: 'Child 2',
                        id: 'child2',
                    },
                    {
                        topic: 'Child 3',
                        id: 'child3',
                    },
                    {
                        topic: 'Child 4',
                        id: 'child4',
                    }
                ]
                    }, 
                ],
            }
         ,
        summaries:[{
            id: 's1',
            text: 'Summary',
            parent: 'child0',
            start: 0,
            end: 2
        }

        ]
    }}
 />

## サマリーデータ

```typescript
export interface Summary {
    id: string;
    text: string; 
    parent: string; 
    start: number; 
    end: number;
}
```

サマリーデータは以下のフィールドを含みます：

- `id`: ユニークID
- `text`: サマリーテキスト
- `parent`: サマリーが属するノードのID
- `start`: サマリーが開始する子ノードのインデックス
- `end`: サマリーが終了する子ノードのインデックス

:::warning

サマリーの親ノードはルートノードにすることはできません。

:::
